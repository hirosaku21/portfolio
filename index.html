保有スキルSkills

職業訓練校でのカリキュラムとは別に、Udemyや書籍などを用いて学習しております。

また、実践的なスキルを身に付けるためにはプロ目線のフィードバックが重要と考え、メンターサービスも活用しました。

  • コーディング（HTML/CSS）

    セマンティックなHTMLと「予測しやすい」「再利用しやすい」「保守しやすい」「拡張しやすい」CSS設計を意識しています。設計手法としては、BEMを主に使っています。また、CSSのコーディングにはSCSSを用いて、mixinや変数管理等で効率化を図っています。

    Visual Studio Code / DreamWeaver

  • デザイン

    設定した目的や世界観を表現するデザインを心掛けています。また、ユーザーに不要な違和感を与えないように、デザインの4原則も意識します。さらに、SEOライティングの経験から、効果的なデザインの重要要素である”言葉”に関しても検証を重ねています。

    Illustrator / Photoshop / XD / Figma

  • プログラミング

    主にjQueryとプラグインをカスタマイズして、基本的なアニメーションを実装可能です。また、PHPとDBを用いて、カート機能/お問い合わせ/管理画面などを開発できます。WordPressについては、オリジナルテーマの作成経験があります。

    JavaScript（jQuery）/ PHP / WordPress

よくある質問Questions

Q.

Web制作のどの分野が好きですか？

A.

サイトの目的達成に繋がるコーディングとデザインの「設計」を組み立てるプロセスに最も興味があります。また、フロントエンド領域の学習も継続していきたいです。

Q.

どのようなキャリアを考えていますか？

A.

現場のことを理解した上で、将来的にはPMをしたいです。そのために、コーディング/デザイン/フロントエンドのスキル習得に加えて、チームでの制作ノウハウを身に付けたいです。

Q.

休日はどのように過ごしていますか？

A.

一人の時間は、筋肉を育てることに勤しんでおります。成長を実感できるように、持ち上げたダンベルの重さや回数を記録する癖があります。直近の目標は、腹直筋にdisplay:gridを取り戻し、三角筋をtransform:scale()させることです。
